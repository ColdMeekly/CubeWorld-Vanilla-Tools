# CubeWorld-Vanilla-Tools
Collection of tools I made to help with managing cubeworld vanilla.
## list_players.py
Matches users against a `whitelist.txt` in the same directory, with the format being `ip=alias` once per line.
e.g. 
```
127.0.0.1=Admin
192.168.1.2=Joe
...
```
